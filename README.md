# Hospital_Management_sys_SQL
# 🏥 Hospital Management System | SQL Project

## 📌 Project Overview

This project demonstrates SQL-based data analysis on a **Hospital Management System** dataset. The objective is to solve real-world healthcare business problems using SQL queries ranging from beginner to advanced level.

The project covers data retrieval, filtering, aggregation, ranking, window functions, Common Table Expressions (CTEs), and analytical queries to generate meaningful insights from hospital data.

---

## 🎯 Project Objectives

- Analyze hospital patient records using SQL.
- Practice real-world SQL interview questions.
- Perform healthcare data analysis.
- Demonstrate proficiency in SQL concepts from basic to advanced.
- Build a strong portfolio project for Data Analyst roles.

## 📈 Project Highlights

This SQL project answers business questions such as:

- Retrieve all patient records.
- Find senior citizens admitted to the hospital.
- Count patients department-wise.
- Identify top treatment costs.
- Calculate average treatment expenses.
- Find the busiest departments.
- Analyze doctor-wise patient counts.
- Identify the most common diseases.
- Compare treatment costs across departments.
- Find patients with the longest hospital stays.
- Rank departments based on revenue.
- Generate monthly admission reports.
- Analyze payment methods.
- Evaluate insurance coverage.
- Discover trends using window functions.
